Nepali Unicode Keyboard (Romanized)

This is the official Nepali Romanized Unicode Keyboard Layout. Unlike phonetic converters that guess your words, this tool uses a strict 1-to-1 keystroke mapping (e.g., typing "k" outputs "क", typing "K" outputs "ख"). This is the required layout for official government typing, Lok Sewa exams, and professional document drafting.

Keyboard Layout Reference

Press Shift to type the top character on each key.

Special Joint Characters

  • क + ् + ष = क्ष
  • त + ् + र = त्र
  • ज + ् + ञ = ज्ञ
  • त + ् + त = त्त
  • द + ् + ध = द्ध
  • द + ् + व = द्व
  • श + ् + र = श्र
